# 🧾 Description de l’application

## 🌍 Nom du projet

**Walee – Solution SaaS de gestion intelligente pour entreprises africaines**

---

## 💡 Concept

**Walee** est une application web moderne tout-en-un conçue pour permettre aux **entreprises (PME, commerces, structures professionnelles)** de gérer efficacement leurs activités au quotidien :
clients, ventes, stocks, employés, finances et statistiques avancées.

L’application fonctionne en mode **multi-entreprises (SaaS)** :
un utilisateur peut créer plusieurs structures (entreprises) et inviter ses collaborateurs en leur assignant des rôles spécifiques (caissier, gestionnaire de stock, administrateur, etc.).

---

## 🚀 Objectifs

* Centraliser la gestion complète d’une entreprise (stock, caisse, facturation, RH, suivi financier).
* Offrir une **vue en temps réel** sur les performances via des tableaux de bord précis.
* Faciliter la **collaboration interne** grâce aux sous-comptes et permissions.
* Fournir une **solution moderne, esthétique et fluide**, accessible depuis ordinateur, tablette ou mobile.
* Donner aux dirigeants des **statistiques au millimètre près** pour une meilleure prise de décision.

---

## ⚙️ Fonctionnalités principales

### 👥 Gestion des utilisateurs

* Création de compte principal (entrepreneur ou société).
* Gestion de **plusieurs entreprises** par le même utilisateur.
* Création de **sous-comptes** (employés, caissiers, gestionnaires, etc.) avec rôles personnalisés.
* Authentification via **Supabase Auth** (email, OTP, Google, etc.).

### 🏢 Gestion d’entreprise

* Création et paramétrage de structures : logo, devise, taxes, adresses, etc.
* Gestion du personnel : employés, rôles, permissions, affectations.
* Suivi de l’activité de chaque entreprise séparément (multi-tenant).

### 💰 Module Caisse & Facturation

* Gestion complète de la **caisse** (entrées, sorties, soldes).
* Création de **factures, devis et reçus imprimables** (PDF).
* Gestion multi-caisses (plusieurs points de vente).
* Historique et rapports financiers.

### 📦 Gestion des Stocks

* Suivi des articles, quantités, alertes de seuil.
* Entrées et sorties automatiques liées aux ventes/achats.
* Rapports d’inventaire.

### 🧾 Comptabilité & Statistiques

* Statistiques de ventes, marges, bénéfices nets, etc.
* Graphiques en temps réel (ventes par jour, par caissier, par produit).
* Exports Excel / PDF.

### 🔔 Notifications & Historique

* Journal d’activité (audit log).
* Alertes automatiques : seuils de stock, dettes clients, échéances.
* Notifications en temps réel via Supabase Realtime.

## 🧱 Stack Technique

| Élément          | Technologie                                 |
| ---------------- | ------------------------------------------- |
| Backend          | **Django (Python)**                         |
| Frontend         | **Django + HTMX + Alpine.js + TailwindCSS** |
| Base de données  | **Supabase (PostgreSQL)**                   |
| Authentification | **Supabase Auth**                           |
| Déploiement      | Render / Vercel / Railway                   |
| API externe      | REST/GraphQL Supabase                       |
| PDF & Export     | WeasyPrint / Pandas / OpenPyXL              |

## 🧩 Style UX / UI

| Composant                   | Style recommandé                                                            |
| --------------------------- | --------------------------------------------------------------------------- |
| **Boutons**                 | arrondis (`rounded-2xl`), ombrés (`shadow-sm`), animation `hover:scale-105` |

| **Formulaires**             | champs espacés, labels clairs, feedback couleur (rouge/vert)                |
| **Navigation**              | barre latérale minimaliste + icônes (Lucide / Heroicons)                    |
| **Tableaux de bord**        | cartes colorées (stats, ventes, alertes) + mini graphiques                  |
| **Modals / Popups**         | coins arrondis, fond semi-transparent sombre                                |
| **Mode sombre (optionnel)** | inverser fond clair ↔ sombre, accent toujours bleu roi                      |

## 🪄 Ambiance visuelle

* Interface **moderne, fluide, professionnelle**, inspirée de Notion / Stripe Dashboard / Odoo.
* Priorité à la **lisibilité et à la rapidité** d’action.
